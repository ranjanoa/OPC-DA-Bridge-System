Industrial OPC DA to InfluxDB BridgeA bidirectional gateway between legacy OPC DA servers (WinCC, Kepware, RSLinx) and InfluxDB 2.x.FeaturesIngestion (kiln1): High-speed read from OPC DA to InfluxDB.Actuation (kiln2): Watcher loop that writes InfluxDB fields back to PLC tags.Web Dashboard: Built-in UI for tag discovery and mapping.Persistence: Automatic configuration saving via JSON.Deployment RequirementsWindows 10/11 or Windows Server.OPC Core Components must be installed on the target machine.Application must run as x86 for DCOM compatibility.Build InstructionsRun build_and_publish.bat to generate a self-contained executable in the /publish folder.
